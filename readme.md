# RGUI

[![Join the chat](https://badges.gitter.im/rguijs/RMMV.svg)](https://gitter.im/rguijs/RMMV)

 RGUI is useful, powerful Reactive Graphical User Interface framework.

## Installation

```
npm install rgui
```
or
```
npm install rgui -g
```
